# Legacy Directory

This directory holds historical compatibility modules and exploratory utilities that are not part of the active SharpSV stage-1/stage-2/stage-3 pipeline.

- `Features.py`, `Init_Image.py`, `net.py`: old compatibility aliases mapped to the current stage-2 package modules.
- `FILE_CLASS.py`, `VCF_CLASS.py`, `pyalign.py`: archival utilities preserved for reference.

The production pipeline entrypoint is `SharpSV.py`.
